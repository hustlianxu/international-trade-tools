# 02 - 语音识别方案选型

## 一、需求

- **语言**：西班牙语（含西班牙/墨西哥/阿根廷/哥伦比亚等口音）+ 普通话（含口音）
- **正确性**：必须高，否则误解客户意思
- **硬件**：MacBook Air M3（16GB）主力机 + 低配 Windows 11 台式机
- **预算**：月费 < 100 元
- **用量**：日均 20-30 条语音，每条 10-60 秒，月约 500 分钟

## 三、详细方案

### 方案 A：本地 MLX Whisper（M3 首选，0 元）

**为什么选 MLX 而非 whisper.cpp？**
- MLX 是苹果官方的机器学习框架，针对 Apple Silicon 统一内存架构优化
- MLX Whisper 比 whisper.cpp 快约 1.78x（[实测来源](https://notes.billmill.org/link_blog/2024/08/mlx-whisper.html)）
- 8bit 量化模型内存占用减半，M3 Air 16GB 跑 medium 无压力

**M3 实测性能**：

| 模型 | 1分钟音频处理耗时 | 内存 | 西语 WER |
|---|---|---|---|
| tiny | ~1.5秒 | 1GB | 10-15% |
| small | ~5秒 | 2GB | 6-9% |
| **medium-8bit** | **~7秒** | **2GB** | **4-6%** |
| large-v3 | ~60秒(接近实时) | 10GB | 2.7% |

**推荐配置**：`mlx-community/whisper-medium-mlx-8bit`
- 7秒处理1分钟音频，完全满足"按一下转写一条语音"的实时体验
- WER 4-6% = 每 20 个词错 1 个，外贸业务可用
- 完全免费 + 离线可用 + 客户语音不上云（隐私好）

**提升口音识别准确率**：
- `initial_prompt` 注入外贸热词：`"cotización, pedido, envío, muestra, pago, factura proforma, FOB, CIF, LC"`
- 净音环境 WER 4-6%；强口音+噪声升至 10%，可升级 large-v3

### 方案 B：火山豆包 ASR 2.0（跨平台首选，~7 元）

**为什么选豆包？**
- Seed-ASR 多语种模型，西语+中文原生支持，免切语种
- 价格极低：0.8 元/小时 = 0.0133 元/分钟，500 分钟 = 6.7 元
- 国内访问快，无需代理
- 纯云端，Mac/Windows/Linux 通用

**接入**：HTTP API（提交音频 → 轮询结果，通常 1-5 秒返回）

### 方案 C：gpt-4o-mini-transcribe（最高准确率，~11 元）

- $0.003/分钟 ≈ 0.0216 元/分钟，500 分钟 = 10.8 元
- 99+ 语种，西语 WER 约 2.7-3%（优于原版 Whisper）
- 需 OpenAI API 访问能力（代理或中转）

## 四、西语口音识别准确率

Whisper 对各西语变体支持（[来源](https://starwhisper.ai/lang/spanish-dictation-software.html)）：

| 口音 | 表现 | 说明 |
|---|---|---|
| 西班牙本土(Castilian) | 优秀 | c/z 咬舌音正确处理 |
| 墨西哥 | 优秀 | 训练语料最多 |
| 阿根廷(Rioplatense) | 优秀 | ll/y 的 "sh" 音 + voseo 正确识别 |
| 哥伦比亚 | 优秀 | 发音平缓 |
| 智利 | 良好 | 极快口语需后处理 |
| 加勒比 | 良好 | s 弱化/辅音脱落可识别 |

**结论**：标准外贸业务西语（询价/报价/物流/付款）**不需要 fine-tune**，medium 模型即可。

## 五、最终推荐

| 业务员画像 | 推荐方案 | 月费 |
|---|---|---|
| **M3 主力机（用户场景）** | **方案 A：MLX Whisper medium** | **≈1 元** |
| Windows 低配台式机 | 方案 B：火山豆包 | ≈8 元 |
| 客户多在拉美，要求最高 | 方案 A+B 混用（本地优先，失败回退云端） | ≈5 元 |

**本仓库默认配置**：`asr.engine: mlx_whisper`（M3 优先），Windows 用户改 `volcengine`。

## 六、多模态 LLM 直接处理语音？

**不推荐**。对比：
- ASR(豆包) + DeepSeek = 7 + 0.2 = **7.2 元/月**，准确率与多模态几乎一致
- GPT-4o audio 原生 = **50-100 元/月**，性价比低

**最优路径**：语音 → Whisper/豆包转文字 → DeepSeek 理解，总成本 1-12 元/月。
